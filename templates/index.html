Bubble Sort

Bubble sort keeps checking pairs and swapping them if they are in the wrong place it slowly pushes the bigger numbers to the end and the smaller ones to the front

Time: O(n²) Space: O(1)

Selection Sort

Selection sort looks through the list finds the smallest number and puts it where it should be it repeats this until everything is in order

Time: O(n²) Space: O(1)

Insertion Sort

Insertion sort builds the list step by step it takes one number at a time and slides it into the right spot like arranging cards in your hand

Time: O(n²) Space: O(1)

Merge Sort

Merge sort breaks the list into smaller pieces sorts each part and then joins them back together it is clean fast and works really well for big lists

Time: O(n log n) Space: O(n)

Quick Sort

Quick sort picks a random number as a pivot and moves smaller numbers to the left and bigger ones to the right then it repeats this on every side until everything is sorted

Time: O(n log n) Space: O(log n)

Heap Sort

Heap sort uses a tree like structure to always keep the biggest number at the top it takes that number out fixes the heap and keeps doing this until the list is sorted

Time: O(n log n) Space: O(1)
