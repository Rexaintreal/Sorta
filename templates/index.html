Bubble Sort

Bubble sort keeps on checking pairs and swapping the pairs if they are in the wrong place it slowly pushes the bigger numbers to the end and the smaller numbers to the front

Complexity
Time: O(n²) Space: O(1)

Selection Sort

Selection sort looks through the array or the list and finds the smallest number then it puts the number where it should be and then repeats this until everything is in order

Complexity
Time: O(n²) Space: O(1)

Insertion Sort

Insertion sort builds the array or the list step by step and it takes one number at a time and keeps it into the right spot like arranging cards in your hand and repeats until it is sorted

Complexity
Time: O(n²) Space: O(1)

Merge Sort

Merge sort breaks the array into smaller pieces then sorts each part and then joins them back together it is clean fast and works really well for big lists or arrays

Complexity
Time: O(n log n) Space: O(n)

Quick Sort

Quick sort picks one random number from the list and treats it like the main point then it pushes all smaller numbers to the left side and all bigger numbers to the right side after that it keeps doing the same thing again on each side until the whole list becomes sorted

Complexity
Time: O(n log n) Space: O(log n)

Heap Sort

Heap sort uses a tree like (Binary Tree) structure to always keep the biggest number at the top it takes that number out fixes the heap and keeps doing this until the list is sorted

Complexity
Time: O(n log n) Space: O(log n)
